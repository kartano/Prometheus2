TaggablePSR6ItemAdapter

Implements \Cache\Taggable\TaggableItemInterface

internal

An adapter for non-taggable cache items, to be used with the cache pool adapter.

This adapter stores tags along with the cached value, by storing wrapping the item in an array structure containing both.

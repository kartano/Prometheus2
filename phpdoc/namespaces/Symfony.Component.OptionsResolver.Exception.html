NoSuchOptionException¶

Thrown when trying to read an option that has no value set.

When accessing optional options from within a lazy option or normalizer you should first check whether the optional option is set. You can do this with isset($options['optional']). In contrast to the UndefinedOptionsException, this is a runtime exception that can occur when evaluating lazy options.
